# 智律小管家 - Smart Life Manager

一款智能生活管理助手应用，帮助用户管理日常作息、闹钟提醒和个人目标。

## 🎯 功能特性

- ⏰ **闹钟管理** - 设置起床和日常提醒时间
- 🎯 **场景模式** - 一键切换工作、学习、运动、休息模式
- 👤 **个人中心** - 管理账户信息和偏好设置
- 📚 **记忆库** - 记录学习内容，定期复习巩固
- 📅 **作息表** - 规划每日作息时间
- 📊 **数据统计** - 查看健康与效率数据

## 🚀 技术栈

- HTML5
- CSS3 (Glassmorphism 玻璃态设计)
- JavaScript (ES6+)

## 📱 界面特色

- 炫酷开场动画CG
- 紫蓝色渐变主题
- 玻璃态卡片设计
- 流畅的页面切换动画
- 响应式设计，支持移动端

## 🛠️ 开发

```bash

# 进入项目目录
```
